Welcome to Komlog

Komlog is a flexible and powerful distributed execution platform, aimed at analyzing and processing time series data.

Its purpose is to model your data and enable any number of devices exchange information between them. Besides being a messaging system, Komlog offers some aditional services, like data visualization, anomaly detection and forecast generation, making it a useful platform for any data driven organization or data scientist.

Process Natural Language

Komlog understands natural language, so you can send it unstructured texts, from command line outputs to log files.
